Listing

Ordered List

From this you may create a list as a comparison to each other i.e. which one is better or worse from each other.

Example

Ranked Best of the IVY league

  1. Harward University
  2. Yale University
  3. Princeton University
  4. University of Pensylvania
  5. Cornell university

Unordered Listing

From this you can make list of thos things that are not related to each other.

Example

Grocercy List

  • Meat
  • Rice
  • Breakfast Cereal
  • Honey
  • Salt

Nested List

This is called the list under list means that you can create a list than at any list you detail wise mention it to another list,
